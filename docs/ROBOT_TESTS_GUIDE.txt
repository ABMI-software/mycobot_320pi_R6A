  [2] Demander les angles du robot              ← Sans danger
  [3] Demander le statut                        ← Sans danger

  [6] Mouvement simple - Joint 1                ⚠️  Robot bouge

Avantages :
  ✅ Guidé étape par étape
  ✅ Confirmations de sécurité pour les mouvements
  ✅ Pas besoin de taper les commandes

═══════════════════════════════════════════════════════════════════════
🐛 DÉPANNAGE RAPIDE
═══════════════════════════════════════════════════════════════════════

Le robot ne répond pas :
  1. Vérifier le port série sur la Pi : ls /dev/ttyUSB*
  2. Vérifier les permissions : sudo usermod -a -G dialout $USER
  3. Tester pymycobot directement (voir commande ci-dessus)

Le bridge ne se connecte pas :
  1. Vérifier que bridge_pi.py tourne sur la Pi
  2. Tester la connectivité : ping 10.10.0.218
  3. Tester le port : nc -zv 10.10.0.218 5005

Pas de topic /to_robot ou /from_robot :
  1. Vérifier que bridge_tour est lancé
  2. Vérifier l'environnement : source install/setup.bash
  3. Désactiver conda : conda deactivate
